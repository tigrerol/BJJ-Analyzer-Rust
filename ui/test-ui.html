🚀 Getting Started

Step 1: Start the Backend API

First, ensure the Rust backend is running with API enabled:

cd ../BJJ-Analyzer-Rust
cargo run --features api -- --video-dir /path/to/videos --api-port 8080

Step 2: Start the Web Server

Start a local web server to serve the UI:

# Option 1: Python
python3 -m http.server 3000

# Option 2: Node.js
npx serve . -p 3000

# Option 3: Using npm (if package.json is available)
npm start

Step 3: Access the UI

Open your browser and navigate to:

http://localhost:3000
